# Ashveil — The Broken Veil

**[▶ Play Now](https://dannifaze-code.github.io/Ashveil/)**

Ashveil — The Broken Veil is a mobile-first, browser-based roguelike RPG. Designed for touch controls with full PC keyboard support.

## 🎮 Platform

**Mobile First, PC Second** — Ashveil is built for mobile browsers with responsive touch controls (virtual joystick + action buttons). Full keyboard support (WASD, arrow keys) is available for desktop play.

## 🕹 Controls

### Mobile
- **Virtual Joystick** — Omni-directional movement (bottom-left)
- **Action Buttons** — Attack ⚔, Gather ✦, Interact ◈ (bottom-right)

### PC
- **WASD / Arrow Keys** — Omni-directional movement with fluid acceleration
- **SPACE** — Attack nearest enemy
- **F** — Gather nearby resources
- **E** — Interact with NPCs
- **R** — Enter dungeon (when near entrance)
- **ESC** — Close dialogue
- **Tab** — Cycle UI panels

## ⚔ Features

- **Fluid omni-directional movement** with acceleration and deceleration
- **Animated sprites** — Directional walking and idle breathing animations for player, NPCs, and enemies
- **4 Zones** — Cindergate (town), Emberwood (forest), Mirefen (marsh), Glass Ruins
- **3-chapter story** with branching faction choices
- **4 rival factions** — Cindergate, Roadwardens, Veil Scholars, Ashen Order
- **Deep crafting** — 20 recipes across weapons, armor, trinkets, and consumables
- **Relic dungeons** — 3 multi-floor dungeons with bosses and unique loot
- **Dynamic world** — Corruption system, ecology, world events, and settlement upgrades
- **Class awakening** — Warden, Artificer, or Veilwalker at level 5
- **18 achievements** and persistent save/load

## 🛠 Tech

Pure vanilla JavaScript + HTML5 Canvas 2D. Zero dependencies. Single-file deployment via GitHub Pages.
